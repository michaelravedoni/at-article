Code

Ci-dessous, le code des composants web utilisé dans l'exemple ci-dessus.

at-note

Une note de marge. Elle peut aussi être uniquement "inline". Selon la largeur de l'écran, la note devient exculsivement "inline". Réduisez lalargeur de l'écran pour le constater. Lorsque une note est rajoutée, elle est automatiquement répértoriée en fin de document. Voir la section "Notes" pour le constater.

<at-note>…</at-note>
<at-note type="inline">…</at-note>

at-callout

Un encadré "inline" ou "block" pour un renvoi vers une resource extérieure.

<at-callout>…</at-callout>
<at-callout type="block">…</at-callout>

at-block

Pour une mise en avant d'un contenu. Celui-ci peut-être de type info, exemple ou default. La numérotation est automatique.

<at-block heading="Si eum fuissemus" type="info" caption="true">…</at-block>
<at-block type="example" caption="true">…</at-block>

import { Component, Prop, State } from '@stencil/core';

@Component({
  tag: 'at-block',
  styleUrl: 'at-block.scss'
})
export class AtBlock {

  // Indicate that name should be a public property on the component
  @Prop() caption: boolean;
  @Prop() captionHeading: string;
  @Prop() heading: string;
  @Prop() type: string = 'default'; // default, info, example, warning

  @State() cHeading: string;

  render() {
    if (this.type == 'info') {this.cHeading = 'Info';}
    else if (this.type == 'example') {this.cHeading = 'Example';}
    else if (this.type == 'warning') {this.cHeading = 'Warning';}

{this.cHeading}
) : (null)}
      {this.heading ? (
{this.heading}
) : (null)}
      
      

    );
  }
}
